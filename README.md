# Inventar ilova (kitob ombori)

Django + Bootstrap asosidagi sodda inventar ilovasi.

## Qisqa yo‘riqnoma
- Kutubxonalarni o‘rnating: `pip install -r requirements.txt` (yoki Django 5+ o‘rnatilgan bo‘lsin)
- Migratsiyalar: `python manage.py migrate`
- Serverni ishga tushiring: `python manage.py runserver`

## Izohlar
- Statik fayllar: `static/`
- Shablonlar: `store/templates/store/`
- Standart baza: SQLite (`db.sqlite3`)
